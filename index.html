About Game Hub

Game Hub is a collection of classic games rebuilt using modern web technologies. Each game is implemented with Three.js to provide 3D graphics, smooth animations, and engaging visual effects.

Technology Stack

  • Three.js - 3D graphics library for WebGL
  • Vite - Fast build tool and development server
  • JavaScript - Game logic and interactivity
  • Git - Version control and external project integration

Integration with External Projects

This hub is designed to easily integrate and reference other Git projects. Each game can be:

  • Pulled in as Git submodules for version control
  • Referenced as npm dependencies
  • Cloned as separate folders within the games directory
